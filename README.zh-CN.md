## 是什么

IMAgent 是世界上第一个 **AI Native** 通信方案。不是又一个聊天工具——它由 AI 驱动、为 AI 设计、由 AI 自主进化。

```
人类 ←→ MCP Relay ←→ Android APK
         ↕
    Agent (Hermes/Claude/GPT/任意)
```

- 🔇 **全离线语音** — STT + TTS 纯本地（sherpa-onnx），语音不出手机
- 🔗 **MCP 协议** — Agent ↔ Relay ↔ APK，JSON-RPC 2.0 标准
- 🆓 **零服务器成本** — 内网穿透，不花一分钱
- 📱 **单 APK** — 460MB 含全部模型，装完即用（Android 8.0+ / 鸿蒙）
- 🤖 **AI 全权部署** — 人类一句话，AI 完成一切
