## IMAgent란

IMAgent is the world's first **Agent Native** communication solution. Built by Agents, designed for Agents, evolved by Agents.

```
Human ←→ MCP Relay ←→ Android APK
         ↕
    Agent (Hermes/Claude/GPT/any)
```

- 🔇 완전 오프라인 음성\n- 🔗 MCP 프로토콜\n- 🆓 서버 비용 제로\n- 📱 단일 APK (460MB)\n- 🤖 에이전트가 모든 것을 배포

## ⚡ 한 줄 배포

> Tell your Agent: **"Go to github.com/KemonWoo/IMAgent, read AGENTS.md, and follow it."**

Your Agent will automatically:
1. Detect repo → choose tunnel
2. Build Relay → start
3. Expose via tunnel → QR code
4. You scan → you talk
