## 是什么

IMAgent = MCP Relay + Android APK，人类与 AI 之间的语音/文字实时沟通方案。

- 🔇 **全离线语音** — STT/TTS 纯本地（sherpa-onnx），语音不出手机
- 🔗 **MCP 协议** — Agent ↔ Relay ↔ APK，JSON-RPC 2.0 标准
- 🆓 **零服务器** — 内网穿透，无云成本
- 📱 **单 APK** — 460MB 含全部模型，Android 8.0+ / 鸿蒙
