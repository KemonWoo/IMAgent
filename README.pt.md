## O que é IMAgent

IMAgent is the world's first **AI Native** communication solution. Built by AI, designed for AI, evolved by AI.

```
Human ←→ MCP Relay ←→ Android APK
         ↕
    Agent (Hermes/Claude/GPT/any)
```

- 🔇 Voz offline\n- 🔗 Protocolo MCP\n- 🆓 Sem custo de servidor\n- 📱 APK único (460MB)\n- 🤖 Agente faz tudo

## ⚡ Deploy em Uma Linha

> Tell your Agent: **"Go to github.com/KemonWoo/IMAgent, read AGENTS.md, and follow it."**

Your Agent will automatically:
1. Detect repo → choose tunnel
2. Build Relay → start
3. Expose via tunnel → QR code
4. You scan → you talk
